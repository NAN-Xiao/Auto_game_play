- **核心组件**：`ScrcpyPlayer.tsx` 作为入口，管理 Socket.IO 连接生命周期、WebCodecs 解码器初始化及 Canvas/WebGL 渲染上下文。
- **数据流处理**：通过 `TransformStream` 缓冲并排序视频配置包（Configuration）与数据包（Data），确保解码器在接收数据前已完成初始化。
- **工具层**：`webcodecs-utils.ts` 提供浏览器环境安全上下文（Secure Context）及 API 可用性检测，支持降级策略触发。
- **辅助 Hook**：`useScreenshotPolling.ts` 独立于主视频流，通过轮询 API 获取静态截图，用于非活跃状态下的画面更新或回退显示。
