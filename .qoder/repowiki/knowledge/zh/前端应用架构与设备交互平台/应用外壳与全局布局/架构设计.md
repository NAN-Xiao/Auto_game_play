- 入口层：`main.tsx` 初始化 `ThemeProvider`、`I18nProvider` 与 TanStack Router，挂载到 `#app`。
- 路由层：`routes/__root.tsx` 定义根路由，组合 `NavigationSidebar`、内容区 `Outlet` 与 `Footer`，并注入 `DeviceProvider`。
- 能力层：`lib/theme-provider.tsx` 封装 `next-themes`；`lib/i18n.ts` 与 `lib/i18n-context.tsx` 提供多语言字典与上下文 API。
- 组件层：侧边导航、主题切换、Toast、确认对话框、Markdown 渲染、可拖拽把手等通用 UI 组件。
- 工具层：自定义 Hooks（移动端检测、本地存储、防抖、页面可见性、版本信息）与 SSE 流解析工具 `lib/sse.ts`。
