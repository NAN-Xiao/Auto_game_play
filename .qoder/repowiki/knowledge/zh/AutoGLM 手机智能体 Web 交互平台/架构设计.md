- **入口与部署**：`__main__.py` 负责 CLI 参数解析、端口探测及 `uvicorn` 启动；`server.py` 将 FastAPI 应用与 Socket.IO 服务器通过 `ASGIApp` 组合。
- **接口层**：`api/` 包含多个路由模块（如 `agents`, `devices`, `control`），处理 HTTP 请求并协调底层服务。
- **核心业务层**：
  - **智能体工厂**：`agents/factory.py` 采用注册表模式动态创建不同模型（GLM, MAI, Gemini 等）的智能体实例，统一实现 `AsyncAgent` 协议。
  - **设备管理**：`device_manager.py` 维护单例状态，通过后台线程轮询 ADB 设备，支持 USB/WiFi/远程设备的连接管理与 mDNS 发现。
  - **配置系统**：`config_manager.py` 实现四层优先级（CLI > ENV > FILE > DEFAULT）的配置合并与热重载机制。
- **通信与流媒体**：`socketio_server.py` 处理前端 WebSocket 连接，驱动 `scrcpy_stream` 实现低延迟屏幕画面推流。
- **抽象边界**：定义 `DeviceProtocol` 和 `AsyncAgent` 协议，解耦具体设备实现（ADB/Remote/Mock）与智能体逻辑。
