- **混合通信架构**：后端采用 FastAPI 与 Socket.IO 的 ASGI 混合挂载模式（`AutoGLM_GUI/server.py`），同时提供 RESTful API 用于状态管理与 WebSocket 用于实时视频流及 Agent 事件推送。
- **多形态交付编排**：根目录通过 `pyproject.toml` 定义 Python 包标准，`docker-compose.yml` 提供容器化部署，`electron/` 目录封装桌面应用壳层，`frontend/` 提供 Web 界面，形成“一次开发，多端分发”的架构。
- **统一配置与资源管理**：后端作为核心枢纽，统一管理 ADB 驱动、Scrcpy 服务器二进制文件及多语言提示词，并通过配置管理器向各子模块（Agent、设备控制）注入运行时环境。
