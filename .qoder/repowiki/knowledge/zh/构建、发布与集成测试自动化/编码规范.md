- 所有脚本均遵循 CLI 规范，通过 `argparse` 解析参数并提供 `--help` 说明，且以 `if __name__ == "__main__":` 作为入口保护。
- 路径处理统一使用 `pathlib.Path` 并结合 `Path(__file__).parent.parent` 动态定位项目根目录，确保脚本在任意工作目录下均可正确执行。
- 外部进程调用优先使用 `subprocess.run` 并设置 `capture_output=True` 或 `check=True`，针对 Windows 平台的 Node.js 工具链（如 npm）显式启用 `shell=True` 以兼容执行环境。
