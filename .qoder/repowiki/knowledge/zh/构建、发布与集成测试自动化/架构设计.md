- **构建流水线**：`build_electron.py` 作为总控入口，串联 `uv sync`、前端 `npm build`、ADB 资源下载 (`download_adb.py`)、PyInstaller 打包 (`autoglm.spec`) 及 Electron 分发。
- **发布管理**：`release.py` 负责多源版本号同步（`pyproject.toml`、`electron/package.json`、`README.md`）并自动执行 `git commit/tag`。
- **测试基础设施**：`start_e2e_services.py` 与 `run_harness.py` 通过 `multiprocessing` 编排 Mock LLM、Mock Agent 及后端服务的生命周期，支持基于语义 Golden File 的回归验证。
- **质量门禁**：`lint.py` 统一调度前端的 ESLint/Prettier/TypeCheck 与后端的 Ruff/Pyright 检查。
- **配置驱动**：`autoglm.spec` 定义了 PyInstaller 的隐藏导入、二进制资源（scrcpy-server）及运行时钩子（UTF-8 编码）。
