- **入口与构建**：`main.js` 为应用主进程入口，`preload.js` 提供安全上下文桥接；`electron-builder.yml` 定义多平台打包配置及 GitHub 发布策略。
- **后端集成**：主进程通过 `child_process.spawn` 启动本地 Python 后端（开发模式用 `uv run`，生产模式用打包二进制），并通过 TCP 端口健康检查确保服务就绪后加载前端 URL。
- **安全通信**：采用 `contextIsolation: true` 与 `nodeIntegration: false`，仅通过 `preload.js` 暴露有限的 IPC 接口（如日志读取、应用重启）给渲染进程。
- **运维支撑**：集成 `electron-updater` 实现静默后台更新；`afterPack.js` 钩子在打包后自动修正后端二进制及 ADB 工具的执行权限；`verify-deps.js` 在构建前校验关键运行时依赖。
